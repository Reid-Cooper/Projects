To run the program please follow these steps
-----------------------------------------------------
1. Make sure all files contained within the zip file are in the same directory of your machine
2. Afterwards in VS code or your desired platform navigate (change directory) into the folder housing all 11 of the Projet 2 files (including this README)
3. Paste this line in the terminal: g++ -Wall -Werror -Wpedantic -std=c++17 -g project2Driver.cpp Board.cpp player.cpp Tiles.cpp 
4. Click enter
5. Give the program a few seconds to compile
6. Once you see an a.out file appear in your folder paste thing line into the terminal: ./a.out           
7. Click enter and enjoy the game!
